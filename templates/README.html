  3. From there, L10 Manager opens and you're ready to go.

About the Level 10 Meeting

L10 Manager is built around the EOS/Traction-style Level 10 Meeting - a weekly leadership meeting run in three phases:

Prep Run Review

The agenda itself uses a specific vocabulary you'll see throughout the app: Segue, Scorecard, Rock Review, Headlines, To-Do List, IDS, and Conclude.

Staying up to date

L10 Manager checks for a newer version shortly after it starts and will ask before updating - you can choose to update now, wait, or skip that particular release. Your data in the Data folder is never touched by an update. You can also check any time from the app's Help → Check for Updates... menu.

L10 Manager is under active development - today's app is an early placeholder, with real Prep/Run/Review tools coming soon.
